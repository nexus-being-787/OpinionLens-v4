## Quick Start

To get started with OpinionLens, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nexus-being-787/OpinionLens-v4.git
   cd OpinionLens-v4
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```
